get /ch4
Requests monthly CH4 measurements. (ch4Get)

Query parameters

year (optional)
Query Parameter — Return all CH4 measurements for a given year. default: null
month (optional)
Query Parameter — Return all CH4 measurements for a given month. default: null
gt (optional)
Query Parameter — Return all CH4 measurements with an average ppb reading greater than the supplied value. default: null format: float
lt (optional)
Query Parameter — Return all CH4 measurements with an average ppb reading less than the supplied value. default: null format: float
gte (optional)
Query Parameter — Return all CH4 measurements with an average ppb reading greater than OR equal to the supplied value. default: null format: float
lte (optional)
Query Parameter — Return all CH4 measurements with an average ppb reading less than OR equal to the supplied value. default: null format: float
simple (optional)
Query Parameter — If true, a smaller, simplified version of each CH4 measurement will be returned. default: false
pretty (optional)
Query Parameter — If true, json responses are indented for readability. default: true
limit (optional)
Query Parameter — Maximum number of items to return. default: 10 format: int32
offset (optional)
Query Parameter — Number of items to skip before returning the results. default: 0 format: int32
page (optional)
Query Parameter — Shifts the response data by offset + (limit * page). default: 1 format: int32
